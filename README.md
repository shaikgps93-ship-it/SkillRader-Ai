# 🚀 SkillRadar AI

AI-Powered Career Intelligence Platform built with Streamlit.

SkillRadar AI helps users analyze resumes, discover jobs, predict salaries, identify skill gaps, and build personalized learning roadmaps.

---

## 🌟 Features

### 📄 Resume Tools
- Resume Analyzer
- ATS Resume Score
- Resume Match Score
- AI Resume + Job Match

### 💼 Job Intelligence
- Live Job Search
- Job Recommendation Engine
- Company Analysis

### 📈 Skill Development
- Skills Analysis
- Skill Gap Analyzer
- Learning Roadmap Generator

### 💰 Salary Insights
- Salary Predictor
- Salary Insights Dashboard

### 🤖 AI Features
- AI Career Advisor
- AI Chatbot

### 📊 Analytics

- Career Metrics

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/shaikgps93-ship-it/SkillRader-Ai.git
```

Move into the project:

```bash
cd SkillRader-Ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
